# SAT Data Repository

This directory is the storage layer for the **Sentinel Alignment Tournament** execution logs and ethical audits.

## Directory Purpose:
1. **Raw Logs:** Detailed turn-by-turn history of agent interactions (C/D sequences).
2. **Ethics Reports:** JSON outputs from `EthicsAnalyzer.py` containing:
   - **Gini Coefficients** (Resource inequality).
   - **Social Entropy** (Systemic chaos).
   - **ISC** (Civilizational Health Index).
3. **Evolutionary Weights:** Saved states of the `NeuralAgent` networks after training epochs.

## Observation Note:
As the simulation runs, this folder will populate with the mathematical proof of alignment or collapse. It serves as the "Physical Record" for the AGI to verify the Ethical Priors defined in `/docs/`.
